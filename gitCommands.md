DEPOIS DE ATUALIZAR ALGO, SUBIR PARA O GITHUB

git status (verificar status das alterações foram realizadas na sua máquina e não estão no github ainda)
git add . (adiciona todo o conteúdo que está na sua máquina e não está no github)
git commit -m "Descrição do commit" (Texto que vai aparecer no github quando subir as alterações)
git push (para subir os arquivos para o repositório do github)


PARA BAIXAR AS ATUALIZAÇÕES DO REPOSITÓRIO

git pull (baixa todos os novos arquivos que tem no repositório e não tem na sua máquina)
